# Stocks Share

## Usage
```
$ git clone https://github.com/karanj798/stocks_share.git
$ cd stocks_share
$ npm install
$ node app.js
```
The app is live on `localhost:3000`.

## Demo
Live Mirror available at [https://stockshare-db.herokuapp.com/](https://stockshare-db.herokuapp.com/)

## FAQ
